# Domain Controller Installation

## Objective

Deploy a Windows Server virtual machine that will function as the Domain Controller (DC-01) for Project Enterprise.

---

## Environment

- Hypervisor: VMware Workstation
- Operating System: Windows Server
- Host System: Windows Gaming PC
- Administration: Remote Desktop from MacBook Neo

---

## Major Tasks Completed

- Installed Windows Server
- Assigned the server name (DC-01)
- Configured networking
- Installed Active Directory Domain Services (AD DS)
- Installed DNS
- Promoted the server to a Domain Controller
- Verified domain functionality

---

## Outcome

DC-01 serves as the central authentication and directory services server for the Project Enterprise lab. It provides identity management through Active Directory and name resolution through DNS.
