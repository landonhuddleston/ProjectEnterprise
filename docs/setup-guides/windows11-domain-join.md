# Windows 11 Domain Join

## Objective

Join a Windows 11 Enterprise virtual machine to the Project Enterprise Active Directory domain.

---

## Environment

**Client**

- Windows 11 Enterprise
- VMware Workstation

**Domain Controller**

- Windows Server
- Active Directory Domain Services
- DNS

---

## Tasks Completed

- Installed Windows 11 Enterprise
- Configured networking
- Verified DNS connectivity
- Joined the workstation to the Active Directory domain
- Verified authentication
- Confirmed the workstation appeared in Active Directory

---

## Outcome

The Windows 11 workstation successfully joined the domain and could authenticate against Active Directory, allowing centralized management within the Project Enterprise environment.
