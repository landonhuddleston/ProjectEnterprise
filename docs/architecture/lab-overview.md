# Project Enterprise Lab Overview

## Objective

Project Enterprise is a hands-on Windows enterprise lab designed to develop practical IT administration and cybersecurity skills through real-world configuration, troubleshooting, and documentation.

## Current Environment

### Domain Controller

- DC-01
- Windows Server
- Active Directory Domain Services (AD DS)
- DNS

### Client

- Windows 11 Enterprise
- Domain Joined

## Remote Administration
