# Project Enterprise

## Overview

Project Enterprise is my personal enterprise IT and cybersecurity home lab.

The goal is to simulate a small business network while developing hands-on experience with technologies commonly used by enterprise IT departments and Security Operations Centers (SOCs).

---

## Technologies

- Windows Server
- Active Directory
- DNS
- Windows 11 Enterprise
- VMware Workstation
- Remote Desktop
- TCP/IP Networking
